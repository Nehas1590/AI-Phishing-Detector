# 🛡️ AI Phishing Detector

> An intelligent phishing email detection web application built using **Python** and **Flask** to identify suspicious emails through keyword analysis, malicious URL detection, fake domain detection, and urgency-based social engineering analysis.

## 🌐 Live Demo

🔗 https://neha1590.pythonanywhere.com

---

# 📌 Project Overview

AI Phishing Detector is a web application that helps users identify potentially malicious phishing emails using multiple detection techniques. It analyzes email content, calculates a phishing risk score, generates an AI confidence level, and provides detailed detection results through a simple and interactive interface.

- 🔍 Keyword-based phishing detection

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Nehas1590/AI-Phishing-Detector.git
```

### Move into Project Folder

```bash
cd AI-Phishing-Detector
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

# 🚀 How It Works

1. User enters email content.
2. The application checks for suspicious keywords.
3. URLs are analyzed.
4. Fake domains are detected.
5. Urgency-based phishing patterns are identified.
6. A phishing risk score is calculated.
7. AI confidence is displayed.
8. Results and scan history are shown.
